## ‚öôÔ∏è Installation & Setup

### 1. Install Python
This course was built using **Python 3.14.1**.
Ensure you have Python 3.14.1 (or newer) installed. You can download it from [python.org](https://www.python.org/downloads/).

### 2. Install Required Libraries
Most projects use built-in Python libraries. However, **Project 7 (Data Pipeline)** requires `requests` and `pandas`.

Open your terminal (or PyCharm Terminal) and run:

```bash
pip install requests pandas
```

## üöÄ How to Run the Code

You can run any script directly from your terminal or IDE.

**1. Open your terminal** (Command Prompt on Windows, Terminal on Mac/Linux).

**2. Navigate to the folder** where you extracted these files.
```bash
cd path/to/folder
```

**3. Run a specific project** by typing `python` followed by the file name.

Example (Running the Casino Game):
```bash
python 2_guess_number.py
```

Example (Running the Data Pipeline):
```bash
python generate_csv.py
```
